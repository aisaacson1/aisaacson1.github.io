How Latitude Affects Weather!

Project Summary

The purpose of this project was to determine what affect if any the latitude of a location has on various weather patterns.


In order to accomplish this a minimum of 500 cities were randomly selected on October 23, 2019, and weather data was gathered through an API. The results of this data is contained within this site.
